Cuenta el número de módulos instalados en el sistema.

Mediante la exploración del archivo /proc/modules, se cuenta la cantidad de drivers instalados.

Returns
La cantidad de módulos del kernel instalados actualmente.
void getData ( struct Modulo  modulos[],
int  lines 
)

Obtiene de el directorio /proc información sobre los drivers en el sistema.

Explorando el archivo /proc/modules se va guardando en la estructura modulos[] la información de cada uno. Puntualmente se guarda su nombre, su tamaño, la cantida de veces que fue montado, y su estado actual.

Parameters
modulos[]Estructura de datos donde se guardará la información de los módulos instalados.
linesCantidad de módulos instalados en el sistema, usado para iterar sobre el archivo.
