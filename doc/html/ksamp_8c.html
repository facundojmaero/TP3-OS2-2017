Detailed Description

Pagina del webserver dedicada a mostrar información del sistema.

Archivo con la lógica y presentación de la página del servidor dedicada a mostrar módulos del kernel instalados, subir un módulo propio, instalarlo y desinstalarlo.

Facundo Maero

Parsea un archivo en busca de una cadena de caracteres.

Recibido un path, una cadena a buscar y delimitadores, busca en el archivo y guarda la cadena en buff. Si no puede abrir el archivo termina el programa. Busca iterativamente hasta encontrar la primer ocurrencia del string buscado y la recorta en base a los delimitadores.

Parameters
fpFile Descriptor, usado para abrir un archivo como lectura
path[]Cadena de caracteres con la ruta en la que se buscara el archivo
before[]String con el delimitador que se encuentra antes de la cadena deseada
after[]String con el delimitador que se encuentra despues de la cadena deseada
buffPuntero a char donde se almacena el resultado
